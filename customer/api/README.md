# Customer, Order and Product API

Create, Read, Update, Upload, Download and Delete (CRUUDD) REST API for managing customer,
product and order.

C.R.U.U.D.D REST API for managing customer.

## Features

- REST API endpoints for managing customer data
- Background job processing with Jobrunr
- Integration with PostgreSQL database
- Integration with Camel - http://localhost:9000/q/camel/dev-console

## Native build

Run `mise` task `native` for a native executable.

## JVM build

Run `mise` task `jvm` for a JVM executable.
